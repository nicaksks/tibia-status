# Endpoint 
| METHOD | ENDPOINT     | QUERY           |
| ------ | ------------ | --------------- |
| `GET`  | `/status`    | `?ip=[string]&port=[string]`|     

# Valores opcionais
`rates e npcs`: Todos os valores são opcionas.<br>
`players`: **unique_players e bots** (`Bots são jogadores conectados em mais de uma conta`) são opcionais
